# Kanu Portfolio — Pure HTML Version 🔐

> 100% HTML + CSS + JS. No PHP. No server needed. Works anywhere!

## 📁 Files

```
kanu-html-portfolio/
├── index.html    ← Everything in one file!
└── README.md
```

## 🚀 How to Use

### Option 1 — Open directly
Just double-click `index.html` in your file manager. Done ✅

### Option 2 — GitHub Pages (Free Hosting!)
1. Create a new repo on GitHub (e.g. `kanu.github.io`)
2. Upload `index.html`
3. Go to Settings → Pages → Source: main branch
4. Visit: `https://kanu.github.io` 🌐

### Option 3 — Netlify (Free Drag & Drop)
1. Go to https://netlify.com
2. Drag the folder onto the dashboard
3. Get a live URL instantly ✅

## 📸 Add Your Photo
- Open the portfolio in browser
- Click the photo area in the About section
- Pick your photo from your device
- Done! (Note: photo saves only in browser session)

## ✏️ Customize
Open `index.html` in any code editor (VS Code recommended) and edit:

| What            | Where in file                     |
|-----------------|-----------------------------------|
| Name, bio text  | About section HTML                |
| Projects        | `const PROJECTS = [...]` in JS    |
| Skills & levels | `const SKILLS = [...]` in JS      |
| Email / GitHub  | Contact section links             |
| Colors          | `:root` CSS variables at top      |
| Typing words    | `const words = [...]` in JS       |
